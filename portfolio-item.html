Memories

MERN-stack social media application

This is a fully responsive online social media website with similar features to popular social media websites like Twitter and Facebook.

1. Functionalities
A list of some of the functions you can perform in the app are:
  • - Creating and Registering user accounts
  • - Authenitcated user login and logout
  • - Creating, Editing and Deleting personal posts
  • - Commenting on other posts

2. Back-end Development
The back-end of the application was built using a REST API with NodeJs as the main programming language. For the database, MongoDB was used to store the schemas. I also utilized JWT to ensure secure authorization and authentication.

3. Front-end Development
The front-end of the application was built using REACT and REDUX as the main front-end frameworks. The application provides secure user authentication with both register and login features.
