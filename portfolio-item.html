MovieHub

MERN-stack application

This is a fully responsive online movie website similar to Netflix I created from scratch to store some of my favourite movies.

Back-end Development
The back-end of the application was built using a REST API with NodeJs as the main programming language. For the database, MongoDB was used to store the schemas. I also utilized JWT to ensure secure authorization and authentication.

The movie clips and trailers are stored using Firebase cloud storage and uploaded to the application using an admin dashboard created using REACT and Context APIs.

Front-end Development
The front-end of the application was built using REACT as the main programming language. The application provides secure user authentication with both register and login features.
